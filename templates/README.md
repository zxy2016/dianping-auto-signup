# 图像模板采集说明

## 需要采集的模板截图

请按照以下步骤手动操作一次，截取关键 UI 元素的截图：

### 1. "免费抽"按钮 (free_lottery.png)
- 打开大众点评 APP
- 进入霸王餐模块
- 在商品列表中，找到任意一个商品右侧的"免费抽"按钮
- 截图保存为 `free_lottery.png`
- **注意**：只需要截取按钮本身，不要包含周围内容

### 2. "我要报名"按钮 (signup_btn.png)
- 点击"免费抽"按钮进入详情页
- 找到页面中的"我要报名"按钮
- 截图保存为 `signup_btn.png`

### 3. "确认报名"按钮 (confirm_btn.png)
- 点击"我要报名"后弹出确认框
- 找到"确认报名"按钮
- 截图保存为 `confirm_btn.png`

### 4. "查看更多活动"按钮 (view_more.png)
- 报名成功后页面会显示成功提示
- 找到页面底部的"查看更多活动"按钮
- 截图保存为 `view_more.png`

### 5. 报名成功提示 (success_hint.png) - 可选
- 报名成功后会显示成功提示
- 截图保存为 `success_hint.png`
- 用于验证报名是否成功

### 6. 验证码相关模板（如果出现验证码）

如果出现滑动验证码，需要采集以下模板：

- **captcha_bg.png** - 验证码背景区域（整个验证码模块）
- **captcha_slider.png** - 滑动滑块
- **captcha_gap.png** - 缺口位置（滑块需要拖动到的目标位置）

## 截图注意事项

1. **尺寸匹配**：确保截图与实际显示尺寸一致（Retina 屏幕注意缩放）
2. **清晰度**：确保截图清晰，不要模糊
3. **完整性**：截取完整按钮，不要切边
4. **状态正常**：确保截取的是按钮的正常状态（非禁用、非按下）
5. **建议工具**：使用系统自带截图工具（Cmd+Shift+4）

## 截图保存

将所有截图保存到 `templates/` 目录下：

```
templates/
├── free_lottery.png
├── signup_btn.png
├── confirm_btn.png
├── view_more.png
├── success_hint.png      # 可选
├── captcha_bg.png        # 可选，验证码检测
├── captcha_slider.png    # 可选，验证码滑块
└── captcha_gap.png       # 可选，验证码缺口
```

## 采集完成后

采集完成后，即可运行 `main.py` 脚本开始自动化报名。

**提示**：如果出现验证码但脚本无法自动处理，脚本会等待 10 秒让你手动处理，然后继续执行。
